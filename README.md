# Getaround — Report Feedback Loop (prototype)

Interactive, single-file mockup of a **positive, visible close-the-loop** on renter-reported
issues (cleanliness / safety / equipment / billing). Every report becomes a warm, status-tracked
ticket — *acknowledged* when flagged, *celebrated* when resolved — with honest closes when the
owner doesn't act, and a forward-trust signal that encourages rebooking. Mock data only, no backend.

**Live demo:** https://whomsfun-ai.github.io/getaround-feedback-loop-prototype/

Open `index.html` in any browser, or use the live link above (embeddable in Notion via `/embed`).
Use the demo console beside the phone to pick a report type, choose how it ends, and advance the loop.
